# Flappy Bird AI

## Features -

- A conventional Flappy Bird Neural Network with NEAT.
- Uses "tanh" for node activation. 
  
## Installation and Setup -

1. **Clone the Repo**:

    ```bash
    git clone https://github.com/YourUsername/fbirdai.git
    cd fbirdai
    ```

2. **Install Dependencies**:

    Using pip:

    ```bash
    pip install pygame
    pip install neat-python
    ```

3. **Run**:

    ```bash
    python main.py
    ```
